## What is it:

It's rust-based task tracker

## How to run for dev:

```bash
yarn add @tauri-apps/cli
```

```bash
cd src-tauri
```

```bash
yarn tauri dev
```

## How to add rust dep:
```bash
cd src-tauri
```

```bash
cargo add <dep_name>
```
